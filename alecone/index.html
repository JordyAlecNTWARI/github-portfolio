// Fonctionnalités

Ce que fait AlecOne

🌐
Viewer 3D interactif
Chaque produit est rendu en 3D avec Three.js. L'utilisateur peut faire tourner l'objet à 360° avant d'acheter.
🔒
Authentification JWT
Connexion sécurisée avec token JWT. Le panier et les commandes sont protégés — seul l'utilisateur connecté y accède.
🛒
Panier temps réel
Ajout au panier instantané avec compteur live dans la navbar. Persistance en base de données côté serveur.
💳
Paiement Stripe
Intégration Stripe Elements pour le paiement par carte. Payment Intent côté serveur, formulaire sécurisé côté client.
⚙️
Back Office Admin
Interface d'administration pour gérer les produits, catégories, commandes et utilisateurs.
🚀
CI/CD GitHub Actions
Pipeline automatisé : build, vérification TypeScript et analyse qualité SonarLint à chaque push.
